Lab Management System

User Management System

One of the major feature is to introduce user authentication and login for the students, teachers and lab instructors. It will help to automate the access system in the laboratory.

Online Request & Approval

The main feature of our project is a student can request for the equipments without being physically present. Teachers and admins can view and approve the request.

Fully Digitalized System

This web application can help to reduce the use of hand-written paper documents. It introduces digital document that keeps the records of the instruments usage by the students.
